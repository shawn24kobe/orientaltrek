# Foshan Furniture Export Website

A static English website for Foshan furniture export, designed for overseas buyers, retailers, and furniture importers. The site uses premium minimalist styling and responsive layouts for GitHub Pages deployment.

## Project Structure

- `index.html` — Homepage with hero image, product categories, style collections, factory/showroom overview, and contact CTA.
- `products.html` — Product center with dual browsing by room and style, plus featured product cards.
- `product-detail.html` — Sample product detail page with images, specs, craftsmanship information, and contact buttons.
- `collections.html` — Style collections page for New Chinese, Light Luxury, and Villa Custom series.
- `showroom.html` — Showroom and factory overview with location and production highlights.
- `about.html` — Company introduction, factory capability, export experience, and certifications.
- `contact.html` — Contact page with WhatsApp, email, WeChat information, and inquiry form.
- `css/style.css` — Shared responsive styling and premium visual structure.
- `js/main.js` — Mobile navigation toggle script.

## Features

- Fixed header navigation with core pages: Home, Products, Collections, Showroom, About Us, Contact
- Responsive design for desktop, tablet, and mobile
- Premium white / light grey / beige visual palette
- High-end overseas buyer friendly layout and copy
- WhatsApp direct contact and business email support
- Static HTML/CSS/JS ready for GitHub Pages

## Deployment

1. Push the repository to GitHub.
2. In repository settings, enable GitHub Pages.
3. Set the source to the repository root branch (usually `main` or `master`).
4. Access the site at `https://<username>.github.io/<repository-name>/`.

## Customization

- Update contact details in all HTML files as needed.
- Replace sample image URLs with actual product and showroom images.
- Add more product detail pages by copying `product-detail.html` and updating content.
- Adjust `css/style.css` for branding or color palette changes.

## Notes

- This website is static and requires no backend.
- Images currently use online sample URLs. For offline or production use, add local image assets and update the `src` values.
- The inquiry form is static and does not submit to a server. You can connect it to a form service or back-end later if needed.
